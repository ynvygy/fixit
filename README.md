About

FixIt is a decentralized platform that allows regular people to report infrastructure problems

Contract deployed on the Calibration Testnet: 0x7c8b0712db1b92fef6821e896c441e16d88aaa2c

How to run / Setup

1. npx hardhat node
2. npx hardhat run ./scripts/deploy.js
3. npm start
